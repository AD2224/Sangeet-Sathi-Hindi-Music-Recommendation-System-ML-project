# 🎵 Sangeet Sathi

A simple Hindi Music Recommendation System based on your mood.  
Type how you feel, and it suggests songs.

---

## 🚀 Features

- Detects emotion from text  
- Supports English + Hinglish  
- Recommends songs based on mood  
- Simple GUI using Tkinter  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- Scikit-learn  
- Tkinter  

---

## ▶️ How to Run

1. Download or clone this project  
2. Keep 'hindi_songs_dataset_.csv' in the same folder  
3. Run:

``bash
python sangeet_sathi_gui.py
